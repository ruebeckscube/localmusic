{% comment %} A re-usable button + modal for creating records with forms. On successful create, it raises the event modal-form-success, with the properties: created_record_name created_record_id Parameters: title_text: the text displayed on the button, and as the title of the modal success_text: the text displayed below the button on success form_html: path to the html for the form htmx_url_name: name of the URL (from urls.py) EXAMPLE: To consume:
{% with title_text="Invite Artist" success_text="Invite sent successfully!" form_html="findshows/htmx/temp_artist_form.html" htmx_url="findshows:create_temp_artist" %} {% include "findshows/partials/modal_form.html" %} {% endwith %}
And in your alpine data definitions: on_temp_artist_create(event) { name = event.detail.created_record_name id = event.detail.created_record_id //further processing } modal based on https://alpinetoolbox.com/examples/modal {% endcomment %}
