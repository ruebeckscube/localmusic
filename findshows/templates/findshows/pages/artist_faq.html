Artist FAQ

How do I…

{% accordion_element title="…create an artist profile/get access to post shows?" %} There are two ways! If you know somebody who already has artist access, they can invite you via their Settings page (or via the concert creation page if you have a show coming up together). If not, go to your Settings page and click Request Artist Access. Enter your artist name and some way for mods to contact you/verify that you are the artist you're claiming to be. {% endaccordion_element %} {% accordion_element title="…post a show?" %} Log in; from the menu, go to My Shows and click Create new show listing. {% endaccordion_element %} {% accordion_element title="…edit a show I'm a part of?" %} Only the user who initially created a show can edit it. Log in; from the menu, go to My Shows. If you created a show, it should have an Edit button next to it. If you just want to edit your artist info it appears on the concert listing, you can do that by editing your artist profile. {% endaccordion_element %} {% accordion_element title="…delete/cancel a show I'm a part of?" %} Only the user who initially created a show can cancel it. Log in; from the menu, go to My Shows. If you created a show, it should have a Cancel button next to it. The concert's detail page will now display prominently the fact that it is cancelled, and it will no longer show up anywhere else (search, artist pages, emails). You cannot delete a show. {% endaccordion_element %}
